The connective layer

ESTAK sits in the middle of the fight.

Every source speaks a different language — a drone streams video, a soldier shares position and voice, a commander needs the overview. ESTAK translates all of it into one picture everyone can act on, and keeps it in sync the instant anything changes.

  • 1
    Unmanned systems push live telemetry, video and status into the shared map.

    Command gets the full common operating picture, SITREPs and fires — without a separate stovepipe.
What it does today

A full operating picture — fielded and working now.

Not a single app, but a unified platform across Android, web and backend. Everyone sees the same thing, the moment it changes.

One platform, everywhere

Android, web and backend working as a single real-time system — one source of truth for every connected client.

A living map

Built on MapLibre with layered basemaps, multiple sources, and precise MGRS / UTM coordinates.

People & comms, together

See who's where — and who's talking, streaming video or typing — fused directly onto the map.

Comms that don't quit

Push-to-talk voice rides a smart, low-bandwidth audio ladder that steps down gracefully instead of dropping the call — over a LiveKit SFU with optional mesh P2P. Built for the contested links unmanned work lives on.

72%

Standards-based symbology

APP-6D / MIL-STD-2525D military symbology — from a curated favourites set to the full catalogue.

Plan, then act

Tasks, teams and SITREPs, plus a Call-for-Fire workflow and UAS fire-request support — planning that turns into action.

Talks to ATAK / WinTAK

Full TAK / CoT interoperability bridges ESTAK with existing ATAK and WinTAK fleets, synced live to every client.

…and it stays in sync

Every marker, message and status update propagates in real time across all connected clients — no manual refresh, no divergence.
